## Configuration:
A json doc must be created within project dir level and the name must be <config.json>. On the other hand, instead
settings.py correction must be done.

- At <config.json>: <br/>
{<br/>
    "SECRET_KEY": "",
    "EMAIL_USE_TLS": "",
    "EMAIL_HOST": "",
    "EMAIL_HOST_USER": "",
    "EMAIL_HOST_PASSWORD": "",
    "EMAIL_PORT": ,
    "AWS_ACCESS_KEY_ID": "",
    "AWS_SECRET_ACCESS_KEY": "",
    "AWS_STORAGE_BUCKET_NAME": ""<br/>
}
